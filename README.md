# Extended Backup Assistant

Chrome扩展备份助手 - 备份、导入和批量安装Chrome扩展程序

## 功能特点

- 📋 **扩展列表获取** - 自动识别所有已安装的Chrome扩展
- 💾 **本地备份** - 导出扩展列表为JSON文件
- 📂 **本地导入** - 从JSON文件恢复扩展列表
- ☁️ **云端备份** - 支持WebDAV协议备份到坚果云
- 🔄 **云端恢复** - 从坚果云恢复最新备份
- 📦 **批量安装** - 一键打开多个扩展的Chrome应用商店页面

## 支持的云存储

- 任何支持WebDAV协议的云存储服务（如坚果云、Nextcloud、OwnCloud等）

## 安装使用

### 1. 安装扩展

1. 打开Chrome浏览器，访问 `chrome://extensions/`
2. 开启右上角的"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择本项目文件夹

### 2. 配置WebDAV

1. 打开扩展弹出窗口
2. 在WebDAV配置中填写：
   - 服务器地址：您的WebDAV服务器地址（如坚果云为 `https://dav.jianguoyun.com/dav/`）
   - 用户名：WebDAV账户用户名
   - 密码：WebDAV账户密码
3. 点击"保存并测试连接"

### 3. 使用指南

#### 获取扩展列表
点击"获取扩展"按钮，自动获取所有已安装的扩展

#### 备份到本地
1. 点击"获取扩展"
2. 点击"导出到文件"，下载JSON备份文件

#### 从本地恢复
1. 点击"从文件导入"
2. 选择之前导出的JSON文件

#### 备份到云端
1. 先点击"获取扩展"
2. 点击"备份到云端"

#### 从云端恢复
1. 点击"恢复最新备份"
2. 扩展将从云端下载最新的备份文件并恢复

#### 批量安装
1. 勾选需要安装的扩展
2. 点击"安装选中扩展"
3. 浏览器将打开多个应用商店页面，点击"添加至Chrome"完成安装

## 权限说明

- `management` - 获取已安装扩展信息
- `storage` - 本地存储配置
- `tabs` - 打开扩展商店页面
- `https://dav.jianguoyun.com/*` - 坚果云WebDAV访问

## 注意事项

1. 扩展备份的是扩展的名称和商店链接，而非扩展的CRX文件
2. 恢复备份后需要重新从商店安装扩展
3. 云端备份会自动创建 `/extbackup` 文件夹
4. 请妥善保管您的坚果云应用密码
